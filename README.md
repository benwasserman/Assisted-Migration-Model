Code and figures for an analysis of a metapopulation model for evaluating the effects of two environmental parameters on evolutionary and demographic responses to environmental change, including testing different strategies for assisted migration.

Important files include:

Assisted Migration Model Parallel.Rmd – As written this conducts one simulation of each possible combination of parameters (landscape type, migration, selection) and saves it. I reran this 30 times.
DataCleaning_NoRescue.Rmd – combines the outputs and shapes data for the no rescue plot ready (Figure 1)
Plots_NoRescue.Rmd – Creates Figure 1
DataCleaning_Rescue.Rmd – shapes the data for the rescue plots
Plots_Rescue.Rmd – Creates the remaining Figures
Regressions.Rmd – Runs the regression analyses
